## :material-rocket-launch: stdapi.ai — OpenAI- & Anthropic-compatible gateway for AWS Bedrock

![stdapi.ai logo](../assets/stdapi_ai_logo.svg){ width=200 }

!!! tip "Run 80+ AI models on AWS Bedrock through OpenAI- and Anthropic-compatible APIs"

    A 14-day free trial is included on the AWS Marketplace.

`stdapi.ai` is a drop-in API gateway that connects your existing OpenAI- and
Anthropic-compatible tools to AWS Bedrock. Change one line — your base URL —
and everything works: LangChain, Open WebUI, Claude Code, Continue.dev, n8n,
and 1000+ other applications.
